API Documentation

The ImageFX Generator provides a RESTful API for generating images programmatically.

Authentication

All API requests require an ImageFX auth token. You can set this in the Settings page.

Endpoints
Generate Images
POST /api/generate
Query Parameters:
  • prompt (required): The text prompt for image generation
  • count (optional): Number of images to generate (1-8, default: 4)
  • seed (optional): Seed for reproducible results
  • aspectRatio (optional): Image aspect ratio (default: IMAGE_ASPECT_RATIO_LANDSCAPE)
    • IMAGE_ASPECT_RATIO_LANDSCAPE
    • IMAGE_ASPECT_RATIO_PORTRAIT
    • IMAGE_ASPECT_RATIO_SQUARE
  • model (optional): Model to use (default: IMAGEN_3_1)
Example Request:
GET /api/generate?prompt=a%20young%20woman%20sitting%20in%20a%20cafe&count=4
Example Response:
{
    "success": true,
    "images": [
        {
            "url": "data:image/png;base64,...",
            "seed": 123456,
            "prompt": "a young woman sitting in a cafe"
        },
        ...
    ]
}
Generate Random Prompt
POST /api/random-prompt
Request Body:
{
    "subject": "a young woman"
}
Example Response:
{
    "prompt": "a young woman casually sitting in a cozy cafe in natural daylight, candid amateur photo, taken with a disposable camera"
}
Get Settings
GET /api/settings
Example Response:
{
    "authToken": "********",
    "defaultCount": 4
}
Update Settings
POST /api/settings
Request Body:
{
    "authToken": "your-auth-token",
    "defaultCount": 4
}
Example Response:
{
    "message": "Settings saved successfully"
}
